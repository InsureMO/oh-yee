# About yee-x

yee-x is a React component library designed specifically for AI application development, aiming to help developers quickly build high-quality AI interactive interfaces.

## Design Philosophy

### 🎯 Focus on AI Scenarios

yee-x focuses on AI application scenarios, providing essential components such as chat bubbles, Markdown rendering, code highlighting, and streaming output.

### 🔧 Highly Customizable

All components provide rich configuration options and custom interfaces, allowing developers to personalize according to their needs.

### 🚀 Ready to Use

Component design follows best practices with reasonable default configurations, enabling developers to get started quickly and reduce configuration costs.

### 💡 Low Coupling Design

Low coupling with specific platforms and frameworks, making it easy to integrate into various projects.

## Core Features

- **Rich Component Library**: Provides various components needed for AI applications
- **TypeScript Support**: Complete type definitions for better development experience
- **Theme Customization**: Supports theme configuration to meet different visual needs
- **Internationalization**: Built-in Chinese and English support, easy to extend to other languages
- **Responsive Design**: Adapts to various screen sizes

## Tech Stack

- React 18+
- TypeScript
- Less/CSS Modules
- Dumi (Documentation Tool)
