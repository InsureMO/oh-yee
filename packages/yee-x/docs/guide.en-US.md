# Quick Start

yee-x is a React component library designed specifically for AI applications. This guide will help you get started quickly.

## Installation

### Requirements

- Node.js >= 16
- React >= 18.0.0
- TypeScript >= 4.5 (optional, but recommended)

### Install via Package Manager

```bash
# Using npm
npm install @rainbow-oh/yee-x

# Using yarn
yarn add yee-x

# Using pnpm
pnpm add yee-x
```
