## Quick Start

### Installation

```bash
# Using npm
npm install @rainbow-oh/yee-x

# Using yarn
yarn add yee-x

# Using pnpm
pnpm add yee-x
```

### Usage

```pure_tsx
import { Bubble, Markdown, CodeBlock } from '@rainbow-oh/yee-x';

function App() {
  return (
    <div>
      <Bubble content="Hello, yee-x!" />
      <Markdown content="# Welcome to yee-x" />
      <CodeBlock code="const hello = 'world';" language="javascript" />
    </div>
  );
}
```
