# Changelog

## Version Release Specification

yee-x follows [Semantic Versioning](https://semver.org/):

- Major version: Incompatible API changes
- Minor version: Backward-compatible functionality additions
- Patch version: Backward-compatible bug fixes

---

## 1.0.0

`2024-01-20`

### 🎉 Initial Release

- 🎊 yee-x officially released
- 📦 Core AI component library provided

### ✨ New Features

- **Bubble**: Chat bubble component with multiple styles and customization
- **Markdown**: Markdown rendering component with code highlighting
- **CodeBlock**: Code block component with syntax highlighting and copy functionality
- **Sender**: Message sender component with multiple input methods
- **History**: History record component
- **Welcome**: Welcome page component
- **Flow**: Streaming output component
- **Process**: Progress processing component
- **Prompts**: Prompt component
- **Commands**: Command component
- **Attachments**: Attachment component
- **Resize**: Resize component
- **IndepWin**: Independent window component
- **AIRenderer**: AI renderer

### 🌐 Internationalization

- Support for Chinese and English
- Complete bilingual documentation

### 📝 Documentation

- Detailed component documentation
- Usage guides and best practices
- Online demos and code examples
