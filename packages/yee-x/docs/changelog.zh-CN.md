# 更新日志

## 版本发布规范

yee-x 遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

---

## 1.0.0

`2026-05-28`

### 🎉 首次发布

- 🎊 yee-x 正式发布
- 📦 提供核心 AI 组件库

### ✨ 新特性

- **Bubble**: 聊天气泡组件，支持多种样式和自定义
- **Markdown**: Markdown 渲染组件，支持代码高亮
- **CodeBlock**: 代码块组件，支持语法高亮和复制功能
- **Sender**: 消息发送组件，支持多种输入方式
- **History**: 历史记录组件
- **Welcome**: 欢迎页组件
- **Flow**: 流式输出组件
- **Process**: 进度处理组件
- **Prompts**: 提示词组件
- **Commands**: 命令组件
- **Attachments**: 附件组件
- **Resize**: 尺寸调整组件
- **IndepWin**: 独立窗口组件
- **AIRenderer**: AI 渲染器

### 🌐 国际化

- 支持中文和英文
- 提供完整的双语文档

### 📝 文档

- 提供详细的组件文档
- 提供使用指南和最佳实践
- 提供在线演示和代码示例

## 如何升级

### 使用 npm

```bash

```

### 使用 yarn

```bash
yarn add yee-x@latest
```

### 使用 pnpm

```bash
pnpm add yee-x@latest
```
