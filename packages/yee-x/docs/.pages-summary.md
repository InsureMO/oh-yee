# yee-x 文档站点页面总览

## 📄 已创建的公共页面

### 1. 首页 (Home)
- **文件**: `index.zh-CN.md` / `index.en-US.md`
- **内容**: 项目介绍、核心特性、快速开始、社区支持
- **特点**: Hero 区域、特性展示、代码示例

### 2. 快速开始 (Guide)
- **文件**: `guide.zh-CN.md` / `guide.en-US.md`
- **内容**: 安装说明、基础使用、完整示例、样式定制
- **特点**: 详细的代码示例和使用说明

### 3. 关于 (About)
- **文件**: `about.zh-CN.md` / `about.en-US.md`
- **内容**: 设计理念、核心特性、技术栈、团队介绍
- **特点**: 展示项目愿景和价值观

### 4. 更新日志 (Changelog)
- **文件**: `changelog.zh-CN.md` / `changelog.en-US.md`
- **内容**: 版本历史、新特性、未来计划
- **特点**: 遵循语义化版本规范

### 5. 常见问题 (FAQ)
- **文件**: `faq.zh-CN.md` / `faq.en-US.md`
- **内容**: 安装、使用、组件、兼容性、性能等常见问题
- **特点**: 分类清晰、提供解决方案

### 6. 资源 (Resources)
- **文件**: `resources.zh-CN.md` / `resources.en-US.md`
- **内容**: 设计资源、开发资源、学习资源、社区资源
- **特点**: 汇总所有相关资源和工具

### 7. 贡献指南 (Contributing)
- **文件**: `contributing.zh-CN.md` / `contributing.en-US.md`
- **内容**: 贡献方式、开发流程、代码规范、PR 指南
- **特点**: 详细的贡献流程说明

## 🚀 使用方式

启动文档站点：
```bash
cd yee-x
pnpm start
```

构建文档：
```bash
pnpm run docs:build
```
