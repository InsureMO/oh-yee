### 在线工具

- **主题生成器**：[https://theme.yee-c.dev](https://theme.yee-c.dev)
- **图标预览**：[https://icons.yee-c.dev](https://icons.yee-c.dev)
- **色板生成**：[https://colors.yee-c.dev](https://colors.yee-c.dev)

### CLI 工具

```bash
# 安装 CLI 工具
npm install -g @yee-c/cli

# 创建新项目
yee-c init my-app

# 添加组件
yee-c add button alert

# 升级版本
yee-c upgrade
```
