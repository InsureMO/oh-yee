# 关于

## 项目简介

yee-c 是一个现代化、高质量的 React 组件库，专为构建 Web 应用程序而设计。

## 我们的使命

- **提升开发效率**：提供开箱即用的高质量组件，减少重复开发工作
- **统一用户体验**：遵循一致的设计规范，确保产品体验的统一性
- **降低维护成本**：完善的类型定义和文档，降低维护和上手成本
- **促进最佳实践**：推广 React 开发的最佳实践和设计模式

## 技术栈

- **React 18+**：基于最新版本的 React 构建
- **TypeScript**：完整的类型定义支持
- **CSS-in-JS**：灵活的样式方案
- **Dumi**：基于 Dumi 的文档站点生成

## 设计理念

### 简洁至上

我们相信简洁的设计能够带来更好的用户体验。每个组件都经过精心设计，去除不必要的复杂性。

### 可访问性

我们致力于让所有用户都能方便地使用我们的组件，包括使用辅助技术的用户。

### 性能优先

组件的性能优化是我们关注的重点，我们确保组件在各种场景下都能保持流畅。

### 可扩展性

提供灵活的 API 和主题定制能力，满足不同场景的需求。
